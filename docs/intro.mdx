# Введение

Инструкция по настройке интеграции с магазина на Avito с сервисом доставки от СДЭК.
Обратите внимание, для генерации ключей и корректной работы интеграции в Avito необходимо настроить [тариф](https://www.avito.ru/business/tariffs).   

:::info

Данная интеграция предназначена для автоматического создания накладных СДЭК для заказов, оформленных по модели
DBS (доставка силами продавца).
Подробнее про модель продаж можно прочитать в [инструкции](https://support.avito.ru/articles/4234).

:::

Основные особенности интеграции:
- Автоматическое создание накладных СДЭК для заказов на Avito без необходимости ручного переноса данных
- Передача трек-номера СДЭК в Avito

<details>
    <summary>Принцип работы интеграции</summary>

    После настройки, система каждые 15 минут проверяет наличие новых заказов на Avito.

    При обнаружении нового заказа, автоматически создается накладная в СДЭК.

    Накладная появляется в [личном кабинете](https://lk.cdek.ru), откуда ее можно распечатать и наклеить на посылку для
    передачи на доставку.
</details>

# Установка и настройка

Для работы интеграции, не требуется установка дополнительного ПО. Вся настройка происходит через личный кабинет СДЭК.

Для этого необходимо перейти на страницу [настроек интеграции](https://lk.cdek.ru/integration) и выбрать режим **DBS** и вкладку Avito.
После чего нажать кнопку "Добавить магазин"

<Image img={require('./img/add_shop.png')} /> 

Далее необходимо сгенерировать токен Avito и ввести его в настройках интеграции.
Инструкцию по генерации токена можно прочитать на [отдельной странице](./ozon-key.mdx).
